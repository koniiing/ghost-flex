Description for Flex-box

  • justify-content:메인축으로의 정렬;
  • align-items:반대축방향의 정렬(반대축으로의 아이템이 1줄일때)
  • align-centent:반대축방향의 정렬(반대축으로의 아이템이 2줄이상일때)
    -> flex-wrap:wrap으로 설정되어있어야 가능하다.
  • flex-flow: 아래 2개의 속성을 축약해서 쓰는 축약형이다.
  • - flex-direction:플렉스의 메인축 방향 지정(기본:row)
  • - flex-wrap:아이템들의 줄바꿈 허용여부(기본:nowrap)
  • gap : 아이템들의 사이간격지정 축약형
  • - row-gap : 행의 간격
  • - column-gap : 열의 간격

Description for Flex-items

  • flex:grow shrink basis;
  • flex-grow:늘어나는 비율(기본:0)
  • flex-shrink:줄어드는 비율(기본 : 1)
  • flex-basis:메인축으로의 크기
  • align-self:반대축으로의 정렬
